XO game server
======================

**Swagger**: http://127.0.0.1:8080/apidocs

Install packages on local machine
-----------------------------------------------------

    make pipenv

Start local process
-----------------------------------------------------

    make pipenv
    make run-local

Run local tests
-----------------------------------------------------

    make pipenv
    make test-local

Start docker-compose
-----------------------------------------------------

    make start

docker-compose tests
----------------------------------------------------

    make test-docker

stop docker-compose
-----------------------------------------------------

    make stop
